# nest-swagger-redoc-integration

## Description

このリポジトリは以下 3 つを統合して利用するサンプルリポジトリである。

- [Nestjs](https://nestjs.com/)
- [Swagger](https://swagger.io/)
- [Redoc](https://github.com/Redocly/redoc)

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# export redoc html
$ yarn redoc
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
